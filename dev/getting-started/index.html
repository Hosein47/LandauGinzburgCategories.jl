Getting Started With LandauGinzburgCategories.jl

Installation

Refer to the Julia website for details on installing Julia. As soon as you have, start it and run

julia> Pkg.add("https://github.com/tkluck/LandauGinzburgCategories.jl")

to install LandauGinzburgCategories and its dependencies. To test whether it worked, type

julia> using LandauGinzburgCategories.Library

julia> orbifold_equivalence(TwoVariables.A{9}, TwoVariables.D{6})
4×4 Array{ℚ[u,v,x,y],2}:
 0                                             0         -v*x + y                                      -x^2 + u
 0                                             0         x^8 + u*x^6 + u^2*x^4 + u^3*x^2 + u^4 + -v^2  -v*x + -y
 v*x + y                                       -x^2 + u  0                                             0
 x^8 + u*x^6 + u^2*x^4 + u^3*x^2 + u^4 + -v^2  v*x + -y  0                                             0

If you see a matrix factorization, the installation worked!
